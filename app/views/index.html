Location Picker Prototypes

These are location picker widgets that read data from the country register and the territory register.

Demos

Demo 1: basic picker

Demo 2: passport details

Demo 3: country of birth

A location picker is used to collect country of birth as part of passport details.

Code

Code for the location picker can be found on Github.

Development versions

Example 1: Plain JavaScript + plain register

This example uses plain JavaScript to query a public register, and populates a <select> element. It demonstrates one of the simplest ways to query a register and present the results to the user.

Example 2: jQuery Typeahead + plain register

This example uses the corejs-typeahead jQuery plugin and populates it using the same data from Example 1. It demonstrates one way to use an off the shelf component with a public register.

Example 3: jQuery Typeahead + register data file

This example uses the corejs-typeahead jQuery plugin and populates it using a handmade data file, based off of register data. It augments the typeahead plugin and allows it to handle synonyms, translations, endonyms, and territory relationships.

Example 4: jQuery Typeahead + fixes + full register data v1

This example builds upon the previous one by adding progressive enhancement and styling tweaks, both inspired by DWP's implementation. It also moves part of the initial data processing logic server-side, has separate pages for English and Welsh, is compatible with more browsers, has some minor usability fixes, and uses a much larger dataset.

Example 5: Custom Typeahead v0.1.2 + full register data v2

This example uses a version of a custom built typeahead that focuses on accessibility. The styling is almost completely recycled from example 4. The data is an enhanced version of the data from example 4, with additional territories and synonyms.
